# Scss Colors Variables

## Installation

### Install by NPM

``` bash
    npm i scss-colors-variables --save
```

## Usage

Add precompiled scss to your global style scss file

``` scss
    @import 'scss-colors-variables';
```  

or

Add compiled css

``` scss
    @import 'scss-colors-variables/css';
```
